# 🚀 Gemini 2.0 Flash - Quick Start Card

## ⚡ 2-Minute Setup

### 1️⃣ Get API Key
```
https://aistudio.google.com/app/apikey
→ Sign in → Create API Key → Copy
```

### 2️⃣ Add to .env
```env
GEMINI_API_KEY=AIza...your-key-here
```

### 3️⃣ Install & Run
```bash
cd backend
npm install
npm run dev
```

**Done!** ✅
