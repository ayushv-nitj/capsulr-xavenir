## 🎨 Common Design Elements

### Glassmorphism:
```css
background: rgba(15, 23, 42, 0.95);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.1);
```

### Gradient Borders:
```css
border: 4px solid;
border-image: linear-gradient(to right, #fbbf24, #f59e0b) 1;
```

### Neon Glow:
```css
box-shadow: 0 0 20px rgba(168, 85, 247, 0.5),
            0 0 40px rgba(168, 85, 247, 0.3);
```

### Text Gradients:
```css
background: linear-gradient(to right, #fde047, #fb923c);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

## ⚡ Performance

### Optimizations:
- CSS transforms (GPU accelerated)
- RequestAnimationFrame for smooth animations
- Lazy loading of effects
- Cleanup on unmount
- Debounced inputs

### Frame Rates:
- Animations: 60 FPS
- Particle systems: 30-60 FPS
- Canvas (Matrix): 30 FPS

---

## 🎭 Animation Timing

### Entrance:
- Duration: 0.5s
- Easing: spring (damping: 15)
- Delay: 0-0.5s (staggered)

### Loop:
- Duration: 2-4s
- Easing: linear or ease-in-out
- Infinite repeat

### Exit:
- Duration: 0.3s
- Easing: ease-out
- Delay: 5s auto-close
